Raphael.sketchpad

Creates a Raphael SketchPad widget.

Parameters

  1. container Raphael or string
  2. options object

Possible options attributes

  • width number (100)
  • height number (100)
  • input string (null)
  • strokes array (empty array)
  • update_input boolean (false)
